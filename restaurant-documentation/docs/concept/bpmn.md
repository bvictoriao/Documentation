# BPMN

## Диаграмма BPMN

![BPMN Диаграмма](/img/bpmn-diagram.png)

## Краткое описание

BPMN диаграмма иллюстрирует бизнес-процесс онлайн-бронирования столика (Use Case UC-004). Процесс начинается с выбора даты и времени пользователем, включает синхронный запрос к внешней системе ресторана для резервирования, сохранение брони в базе данных и отправку уведомления. Диаграмма также покрывает альтернативные потоки, такие как занятость столика и ошибки внешней системы, а также бизнес-правила, ограничивающие одновременные брони.
